Problem:Train a machine learning model to recognize handwritten digit. The digit must be in english.

Dataset used for training model : The MNIST database of handwritten digits is used for training the model. It has a training set of 60,00 examples, and a test set of 10,00 examples. It is a subset of a larger set available from NIST. The digits have been size-normalized and centered in a fixed-size image.

Model Type : K-NearestNeighbour ML model is used for training perpouse.

What I did : I trained a Machine Learning Model (KNN) and tuned it according to the MNIST dataset. The dataset has 7000 28*28 pixel images of all digits . I trained the model based on the greyscale values of those digits. Then I made a webpage where user can draw digit in a canvas.I take the canvas image resize it into 28*28 , convert into greyscale.Then converting that image into numpy array and feed it to the train model. After that the model predict the value and show it on web page.

Conclusion This model is for only educational purpose.The accuracy of that model highly depends on the image size shape and angle .
